# Hard CTF - Full Cloud Environment Takeover

An attacker already has user-level access to a cloud account. Security alerts indicate logging was disabled shortly after

Your goal is to identify how full control was achieved

---

## Observation

Logs show: - **A new admin user was created** - **IAM policies were modified** - **CloudTrail logging was turned off** - **New virtual machines were deployed**

---

## Question

What **primary attacker objective** is being demonstrated?

---

## Flags (Choose One)

-   **A)** Data exfiltration
-   **B)** Phishing campaign
-   **C)** Persistence and control
-   **D)** Denial of service

---

Correct Flag: **C**
